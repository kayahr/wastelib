wastelib demo

The demo needs access to the original Wasteland data files on your system. Please click the Browse button below and multi-select the following files: . You can also simply select all files in the Wasteland directory. Unneccessary files are ignord. The files are then installed in your browser (IndexedDB) so you only have to do this once.

If you have the Steam version of Wasteland then you may find the files here:

  • Linux: ~/.steam/steam/steamapps/common/Wasteland/rom/data
  • Windows: C:\Program Files\Steam\SteamApps\common\Wasteland\rom\data
